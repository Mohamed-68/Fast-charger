# Bi3wChri - One Product Website

A modern, high-performance landing page for the 240W Folding Charging Cable, designed for the Saudi market.

## Features
- **Modern Arabic Design**: RTL layout with "Tajawal" font.
- **Visuals**: Glassmorphism, smooth animations (Framer Motion), and dark mode aesthetic.
- **Fast**: Built with Vite + React.
- **Responsive**: Fully optimized for mobile and desktop.

## Setup & Run

1.  **Install Dependencies** (if not already done):
    ```bash
    npm install
    ```

2.  **Start Development Server**:
    ```bash
    npm run dev
    ```
    Then open [http://localhost:5173](http://localhost:5173) in your browser.

3.  **Build for Production**:
    ```bash
    npm run build
    npm run preview
    ```

## Project Structure
- `src/components`: UI Components (Hero, Features, Details, etc.)
- `src/index.css`: Global styles (Tailwind v4)
- `public/product`: Product images
